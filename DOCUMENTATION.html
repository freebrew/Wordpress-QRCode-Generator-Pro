WordPress QR Code Generator Pro

Support: contact@brunobrottes.com

WordPress QR Code Generator Pro is a professional plugin for WooCommerce that lets you generate, manage, and track QR codes for your products, categories, shop, and custom URLs. It features beautiful PDF templates, advanced analytics, and a bulk management interface that feels just like native WordPress.

Who is this for?
Store owners, marketers, agencies, and developers who want to boost sales, print professional QR codes, and track real-world engagement.

Features

  • Professional PDF Generation: 2x2 product layouts, ink-efficient color, single-page, print-ready.
  • Multiple QR Types: Product, Category, Shop, Custom URL.
  • Bulk Management: WordPress-style table, checkboxes, bulk actions, filtering, sorting.
  • Smart Duplicate Prevention: Only products without QR codes are shown for new generation.
  • Real-Time Analytics: Track scans, conversions, revenue, and more.
  • WooCommerce Deep Integration: HPOS support, native price formatting, category support.
  • Security: Nonce verification, input sanitization, XSS/CSRF protection.
  • Mobile Responsive: Admin interface works on all devices.
  • Comprehensive Documentation: For both end users and developers.

Installation & Requirements

Requirements

  • WordPress 6.0 or higher
  • WooCommerce 7.0 or higher
  • PHP 7.4 or higher (PHP 8+ recommended)
  • MySQL 5.7+ or MariaDB 10.3+
  • GD extension enabled (for QR image generation)
  • SSL certificate (recommended for secure QR links)

Installation Steps

  1. Unzip the plugin package from CodeCanyon.
  2. Upload wordpress-qr-code-generator-pro to wp-content/plugins/.
  3. Activate via Plugins > Installed Plugins in your WordPress admin.
  4. Go to QR Codes > Dashboard to start generating QR codes.
Note: On first activation, the plugin will create all required database tables and directories automatically.

Admin Interface Guide

Dashboard Overview

Screenshot: Dashboard with Summary Cards
  • Summary Cards: See total scans, visitors, conversions, revenue, and conversion rate at a glance.
  • Generate QR Code Form: Choose QR type, product/category, template options, and size. Only products without QR codes are shown.
  • Performance Table: Manage all QR codes with checkboxes, bulk actions (enable, disable, delete, regenerate), and filters (category, status).

Generating a QR Code

  1. Choose QR type (Product, Category, Shop, Custom URL).
  2. Select the product or category (if applicable).
  3. Pick template options (header, footer, navigation, sidebar).
  4. Choose QR code size (150px-400px).
  5. Click Generate QR Code Template.
  6. Download the PDF or preview the template.
Screenshot: Generate QR Code Form

Managing QR Codes

  • Use checkboxes to select multiple QR codes.
  • Bulk actions: Enable, Disable, Delete, Regenerate.
  • Filter by category or status using dropdowns above the table.
  • Sort columns by clicking headers (product, scans, conversions, revenue, date).
  • Download PDF, view product, or delete via row actions.

Troubleshooting & FAQ

Common Issues

  • Missing GD extension: Enable GD in your PHP configuration.
  • File permissions: Ensure wp-content/uploads/qr-codes/ is writable.
  • WooCommerce not active: Activate WooCommerce before using the plugin.
  • PDF not generating: Check server memory limits and TCPDF installation.

Debugging

  • Enable WP_DEBUG in wp-config.php for error logs (WordPress Debugging Guide).
  • Check the logs/ directory for plugin error logs.
  • Use the System Status page in the plugin admin for diagnostics.
