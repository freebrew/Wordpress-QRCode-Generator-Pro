Generate, track, and analyze QR codes for WooCommerce products with detailed analytics and conversion tracking.

== Description ==

WordPress QR Code Generator is a comprehensive plugin that enables WooCommerce store owners to generate, track, and analyze QR codes for their products and services. The plugin provides detailed analytics on QR code usage, conversion tracking, and engagement metrics.

= Key Features =

* **QR Code Generation**: Generate QR codes for WooCommerce products, categories, and custom content
* **Bulk Generation**: Create multiple QR codes at once with batch processing
* **Real-time Tracking**: Track scans, device information, and user behavior
* **Conversion Analytics**: Monitor purchases and revenue attribution from QR code scans
* **Detailed Dashboard**: Comprehensive admin interface with charts and statistics
* **Customization**: Custom colors, sizes, logos, and error correction levels
* **GDPR Compliant**: Built-in privacy features and data protection
* **Shortcodes & Widgets**: Easy integration with posts, pages, and sidebars
* **Export Data**: Export analytics in CSV, JSON, and Excel formats

= Use Cases =

* Product marketing campaigns
* Print advertising with trackable links
* In-store product information
* Event marketing and promotions
* Social media campaigns
* Email marketing enhancement
* Offline to online conversion tracking

= Requirements =

* WordPress 6.0 or higher
* WooCommerce 7.0 or higher
* PHP 7.4 or higher with GD extension
* MySQL 5.7 or higher

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/wordpress-qr-code-generator/` or install through WordPress admin
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Ensure WooCommerce is installed and activated
4. Go to QR Codes > Dashboard to start generating QR codes

== Frequently Asked Questions ==

= Do I need WooCommerce? =

Yes, this plugin requires WooCommerce for product-related features. However, you can still generate QR codes for custom content without WooCommerce.

= Can I customize QR code appearance? =

Yes, you can customize colors, size, add logos, and set error correction levels.

= Is tracking GDPR compliant? =

Yes, the plugin includes GDPR compliance features and automatically anonymizes data when enabled.

= Can I export analytics data? =

Yes, you can export data in CSV, JSON, and Excel formats with custom date ranges.

= Does it work with caching plugins? =

Yes, the plugin is compatible with popular caching plugins and includes its own caching system.
